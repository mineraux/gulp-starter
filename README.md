Gulp starter
=========

### Introduction

Starter pour pouvoir commencer un site statique sur de bonnes bases.
***Gulp***, ***Browserify***, ***Sass (atomic pattern)***


### Tâches Gulp

**Watch**

Pour lancer le serveur :

```sh
$ gulp
```

**Build**

Pour créer le dossier de distribution:

```sh
$ gulp build
```
